# Pasos para las dependencias que necesito

## Paso 1 | Class validator (versiones estables en Nest 11)

Para los dtos instalar:
`npm install class-validator@0.14.1 class-transformer@0.5.1`

## Paso 2 | Variables de entorno

`npm i @nestjs/config`

## Paso 3 | Contenido estático

`npm i @nestjs/serve-static`

## Paso 4 | Instalar driver de Postgres

`npm install pg`

## Paso 5 | Instalar el ORM

`npm install --save @nestjs/typeorm typeorm`

## Paso 6 | Instalar Passport y el tipadp

`npm install --save @nestjs/jwt passport-jwt`
`npm install --save-dev @types/passport-jwt`

## Paso 7 | Instalar Bcryp

`npm i bcrypt`
`npm install --save @types/bcrypt`

## Paso 8 | Instalar Multer para los files

`npm install cloudinary multer multer-storage-cloudinary`
`npm i -D @types/multer`
